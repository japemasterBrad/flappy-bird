# Clone Flappy Bird
Học sử dụng Unity, Sprite, hơi lạc đề một chút.

## Hướng dẫn cài đặt
Tải về thư mục Release > chạy file `flappy-bird.exe`
Sử dụng chuột trái hoặc phím space để di chuyển chim.
